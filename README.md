1、设定项目路径

2、收集目录信息，设定默认配置

    - 找到模板信息
    - 找到js入口信息
    - 找到发布脚本的位置

3、执行命令

    - start: 启动开发服务器，编译脚本
    - init <name>: 初始化一个页面目录
    - build: 构建生产环境代码
    - publish: 启动发布脚本

4、客户定制配置信息

    - 如果项目目录下有webpack.dev.config.js，放弃默认，直接使用客户定制
    - 如果项目目录下有webpack.pro.config.js  放弃默认，直接使用客户定制
    - 如果目录下有entry.config.js 放弃查找，直接使用客户定制
